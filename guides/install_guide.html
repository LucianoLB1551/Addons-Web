Addons installation tutorial:

Step 1: Downloading the addon

Go to: My addons

Prefixes:

  • [RP] - Resources Pack
  • [BP] - Behavior Pack

Make sure to download the file with the ".mcaddon" or ".mcpack" extension. If the file was downloaded with the ".zip" extension, example: "myaddon.mcpack.zip" you can delete ".zip" extension and should to work.

Step 2: Importing the file into Minecraft

Open the file you want to import

You should see some like this just wait...

When the import proccess finish you should some like this

Congratulations you imported an addon, keep in mind that if you import an addon they already have you will get an error.

Step 3: Enabling the addon

The last step, but pay attention because it is not easy for new players.

Note: The addons in Minecraft Bedrock are not global, that is, you have to activate them in the world you want to use.

So when creating a world (if you already have one you must edit it), after configuring your world as you prefer, go to the experiments section:

and enable all add-on creators options:

After that go to the behaviors section:

And activate the addon you have downloaded.

Now the last step, go to the resource pack section

And activate the addon you have downloaded.

You only have to create the world and enjoy your addon.
